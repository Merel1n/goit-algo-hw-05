# Висновки щодо продуктивності алгоритмів пошуку підрядка

У програмі було реалізовано три класичні алгоритми пошуку підрядка:

Boyer-Moore (BM)

Knuth-Morris-Pratt (KMP)

Rabin-Karp (RK)

Для кожного з двох текстів виконувалось вимірювання часу роботи алгоритмів для:

існуючого підрядка

вигаданого (неіснуючого) підрядка

Замір проводився за допомогою timeit (50 запусків на кожний алгоритм).

---

## Визначення найшвидших алгоритмів

### Текст 1 (memoir_1.txt)

На основі отриманих даних:

Для існуючого підрядка найшвидший алгоритм: Boyer-Moore

Для вигаданого підрядка найшвидший алгоритм: Boyer-Moore

#### Причина:
Boyer-Moore виконує пошук з великими зсувами завдяки "bad character rule", тому працює особливо ефективно на текстах із довгими унікальними послідовностями.

### Текст 2 (memoir_2.txt)

За результатами замірів у другому тексті:

Для існуючого підрядка найшвидший алгоритм: Knuth-Morris-Pratt

Для вигаданого підрядка найшвидший алгоритм: Boyer-Moore

#### Причина:
KMP добре працює в умовах часткових збігів завдяки таблиці префіксів, уникаючи повторних порівнянь.
BM залишається найшвидшим при пошуку неіснуючого рядка.

## Загальний переможець (за всіма тестами)
Найшвидший алгоритм загалом: Boyer-Moore

Boyer-Moore виграє у більшості сценаріїв, оскільки:

- виконує менше порівнянь;

- робить великі зсуви при невідповідностях;

- особливо швидкий на довгих текстах з великим алфавітом.

## Підсумок

*Boyer-Moore*:
_Переваги:_	
- Найшвидший у більшості випадків; 
- Великі зсуви; 
- Ефективний для довгих текстів	
_Недоліки:_
- Складніша реалізація; 
- У найгіршому випадку повільніший за KMP
*KMP*:	
_Переваги:_ 
- Гарантована лінійна складність; 
- Ефективний при повторюваних патернах	
_Недоліки:_
- У середньому повільніший за BM
*Rabin-Karp*:	
_Переваги:_
- Добрий для пошуку багатьох шаблонів; 
- Проста логіка	
_Недоліки:_
- Накладні витрати на хешування; 
- Не найшвидший в одношаблонному пошуку

---

## Висновок

Попри те, що всі три алгоритми коректно виконують пошук, Boyer-Moore виявився найшвидшим у більшості тестів і є найбільш ефективним для великих текстів.
KMP показує найкращий гарантований час у теорії, а Rabin-Karp корисний для задач з множинними шаблонами.
